# BoomerBill Marketing Plan

## Positioning
Measure the real cost of invisible tech labor without accounts, tracking, or lock-in.

## Core Channels
- GitHub: README-led discovery, release notes, changelog proof.
- X/Twitter: short pain-to-solution threads, usage examples, boundary stories.
- Newsletter/site: deeper case posts and hosted beta CTA.

## Content Cadence
- Weekly: one release/changelog post.
- Weekly: two short X posts (problem angle and workflow demo).
- Monthly: one long-form breakdown of insights from usage patterns.

## Funnel and CTAs
- Primary CTA: Try local version.
- Secondary CTA: Join hosted beta waitlist.
- Events: readme_visit, quickstart_start, first_session, week_summary, hosted_waitlist.

## 90-Day Experiments
- Test README hero copy: outcome-first vs feature-first.
- Test GIF placement above quickstart for activation lift.
- Test X posts with data points vs story-based hooks.
